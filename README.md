# 🌐 NetNova

Hey! Welcome to NetNova. This is an interactive web app I'm building to help beginners finally understand computer networking without getting bored to death by dry theory. 

Instead of just staring at blocks of text, you can actually interact with the concepts, see animations, and test what you've learned.

---

## 📖 Why I'm Building This:

Let's face it: learning networking is usually pretty tough because most textbooks just make you memorize endless definitions. 

I wanted to change that. I'm building this platform as a personal project to wrap my own head around these concepts while leveling up my web development skills. My goal is to make a tool that uses:
* Simple, no-jargon explanations
* Cool interactive visualizers
* Quick practice tools and quizzes to test yourself

---

## ✨ Features (doing & Coming Soon)

* 🎨 **Modern Interface:** A clean, dark-themed responsive homepage.
* 🗺️ **Learning Roadmap:** A clear step-by-step path for absolute beginners.
* 🔢 **IP Tools:** An interactive Class Finder and a Public vs. Private IP checker.
* 📦 **Packet Journey:** A visual simulation showing exactly how data travels.
* 🧠 **Quizzes:** Quick interactive check-ins to make sure things actually click.

---

## 🛠 My Tech Stack

Right now, I'm keeping things clean and simple:
* **HTML5** (Structure)
* **CSS3** (Custom properties, animations, and glassmorphism)
* **JavaScript** (For all the interactive logic and tools)

**What I plan to add later:**
* React (Once the vanilla version is rock solid)
* Local Storage (To save quiz scores and roadmap progress)
* Way more advanced interactive tools (like a subnet calculator)
